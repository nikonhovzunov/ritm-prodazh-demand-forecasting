# Формулировка для резюме

**Ритмы продаж 2026** — 16 место из 125, All Cups, прогнозирование 14-дневного спроса.

- Построил CatBoost-модель для 14-дневного прогноза спроса по истории продаж, ценам, промо, остаткам и лагово-роллинговым признакам.
- Реализовал полный авторегрессионный rollout и валидировал решение как многошаговый прогноз на временных фолдах и отложенной выборке.

Расширенная версия для GitHub / интервью:

- Использовал CatBoostRegressor для daily demand forecasting по `nm_id`: календарные признаки, price/promo/stock features, лаги `qty` до 42 дней и rolling statistics за окна 7/14/28 дней.
- Для multi-step прогноза на 14 дней применил autoregressive rollout: предсказания каждого дня добавлялись в историю и использовались для расчета лагов следующих дат.
- Настроил weighted MAE с повышенным весом ошибок на ненулевом спросе и post-processing прогнозов (`clip >= 0`, cap by `prev_leftovers`, integer rounding); лучший подтвержденный CV-вариант — `wMAE 2.761 ± 0.445`.
